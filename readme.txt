================================================================================
                           Todo アプリ 仕様書
================================================================================

概要
----
シンプルな CLI (コマンドラインインタフェース) Todo アプリケーション。
Python で作成され、タスクデータを JSON ファイルに保存します。

ファイル構成
------------
todo.py       - メインスクリプト
todos.json    - タスクデータファイル（自動生成）

要件
----
- Python 3.8+ （推奨: Windows では `py` コマンドを使用）

使い方
------
ターミナルでこのフォルダに移動し、以下のコマンドを実行:

    py todo.py add "タスク内容"        # タスクを追加
    py todo.py list                    # タスク一覧を表示
    py todo.py done <番号>             # 指定番号のタスクを完了にする
    py todo.py delete <番号>           # 指定番号のタスクを削除

操作例
------
    $ py todo.py add "買い物に行く"
    追加しました: 買い物に行く

    $ py todo.py add "レポートを書く"
    追加しました: レポートを書く

    $ py todo.py list
    === Todo リスト ===
      1. [ ] 買い物に行く
      2. [ ] レポートを書く

    $ py todo.py done 1
    完了にしました: 買い物に行く

    $ py todo.py delete 2
    削除しました: レポートを書く

データ形式
----------
todos.json の中身例:
    [
      {"title": "買い物に行く", "done": true},
      {"title": "レポートを書く", "done": false}
    ]

エラー処理
----------
- タスク名が空の場合: "エラー: タスク名が空です。"
- 存在しない番号の場合: "エラー: 番号 X は存在しません..."
- 数値以外の番号: "エラー: 番号は整数で指定してください。"
- 不明なコマンド: USAGE を表示

ローカル化
------------
- メッセージ: 日本語
- 出力エンコーディング: UTF-8 (Windows コンソール対応)
